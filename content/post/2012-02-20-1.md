ubuntu10.10后，在新立得软件包管理器里并没有立得软件包管理器－编辑－－使用任务标记分组软件包－－怎么办？  
  
实际上只需2条命令即可  
  
sudo apt-get install tasksel  
  
任务标记分组  
  
sudo tasksel install lamp-server  
  
安装途中MySQL要求设定账户有密码，这是与操作系统没关系的账户与密码。  
  
怎么样，不错吧。
